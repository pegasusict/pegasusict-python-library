## GOAL
My goal is to write a Python Library which consists of commonly used classes in various Python projects.

### Currently, it holds:
* configuration classes for ini, json, yaml and toml
* .env parser
* arg parser
* Semantic versioning

### (planned) features:
* database interfaces:
    * mysql
    * sqlite
